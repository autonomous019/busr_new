Busr: Smart Way Finding for Transit

Transit Adaptor (transitor) - Find Smart Routes in Transit (Coming Soon!)

Features: Smart Route Finding, Real Time Rider-to-Rider Feedback/Messaging, Commuter Alarm Clock
Platforms: Web (Node.js/HTML5), iOS and Android
API: Busr consumes non-standardized feeds which vary from agency to agency and standardized GTFS Feeds and then presents all feeds in a uniform GTFS based API and for real-time data in GTFS-Realtime to ease transit app development.
